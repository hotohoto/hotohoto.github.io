Here are my favorite eclipse shortcuts, and their IntelliJ counterparts.

* Close current window
  * Eclipse: `Ctrl + W`
  * IntelliJ: `Ctrl + F4`

* Go to definition
  * Eclipse: `F3`
  * IntelliJ: `Ctrl + B`

* Go back to the location where I was at
  * Eclipse: `Alt + ←``
  * IntelliJ: `Alt + N, B`

* Go to the last editing location
  * IntelliJ: `Ctrl + Shift + Backspace`

* Redo
  * Eclipse: `Ctrl + Y`
  * IntelliJ: `Ctrl + Shift + Z`

* Quick fix
  * Eclipse: `Ctrl + 1`
  * IntelliJ: `Alt + Enter`

* Rename
  * Eclipse: `Alt + Shift + R`
  * IntelliJ: `Shift + F6`

* Organize imports
  * Eclipse: `Ctrl + Shift + O`
  * IntelliJ: `Ctrl + Alt + O`

* Show in ...
  * Eclipse: `Alt + Shift + W`
  * IntelliJ: `Alt + F1`

(I am going to add items to this list just for my personal needs.)
